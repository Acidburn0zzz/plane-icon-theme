#Plane-icon-theme

Icons generator, 

- Source icons in = ./src/plane ./src/plane-dark

- Render icons in = ./plane ./plane-dark


## Build
- Is necesary nodejs 7.0 +
- `npm install`
- `npm start`
- `gulp`


## Developer
``` 
$ sudo gulp link 
$ gulp watch
```

## Installation
```
$ gulp
$ sudo gulp copy
```
